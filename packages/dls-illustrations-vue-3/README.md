# dls-illustrations-vue-3

![](https://img.shields.io/npm/v/dls-illustrations-vue-3.svg)

## Installation

```shell
npm i -D dls-illustrations-vue-3
```

## Usage

### As Vue component

```vue
<script setup>
import { IllustrationNotFound } from 'dls-illustrations-vue-3'
</script>

<template>
<figure>
  <illustration-not-found />
</figure>
</template>
```
