# dls-illustrations-vue

![](https://img.shields.io/npm/v/dls-illustrations-vue.svg)

## Installation

```shell
npm i -D dls-illustrations-vue
```

## Usage

### As Vue component

```vue
<template>
<figure>
  <illustration-not-found />
</figure>
</template>

<script>
import { IllustrationNotFound } from 'dls-illustrations-vue'

export default {
  components: {
    IllustrationNotFound
  }
}
</script>
```
